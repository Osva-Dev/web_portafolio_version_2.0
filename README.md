# Osvaldo Ochoa - Web Portafolio

_“El inicio de una etapa enfocada en demostrar compromiso, responsabilidad y crecimiento profesional.”_

Este es mi portafolio personal, actualmente en **fase de borrador**. Está construido con **React + Vite** y estilizado con **Tailwind CSS**. El objetivo es crear una vitrina digital que muestre mi trabajo, mi enfoque técnico y mi evolución como desarrollador web junior.

## 🛠️ Tecnologías utilizadas

- React 19: Interfaz de usuario y componentes.
- Vite: Build tool y desarrollo rápido.
- Tailwind CSS: Estilos utilitarios y responsivos.
- Git / GitHub: Control de versiones y flujo de PR.
