# Portafolio Web - Osvaldo Ochoa

_“El inicio de una etapa enfocada en demostrar compromiso, responsabilidad y crecimiento profesional.”_

Este es mi portafolio personal, actualmente en **fase de borrador**. Está construido con **React + Vite** y estilizado con **Tailwind CSS**. El objetivo es crear una vitrina digital que muestre mi trabajo, mi enfoque técnico y mi evolución como desarrollador web.

## Tecnologías utilizadas

- **React 18** - Biblioteca para construir interfaces de usuario.
- **Vite** - Herramienta de construcción rápida y moderna.
- **Tailwind CSS v4** - Framework de CSS utilitario para estilos rápidos y personalizados.
- **ESLint** - Linter para mantener la calidad del código.
- **Git & GitHub** - Control de versiones y flujo de trabajo con Pull Requests.
